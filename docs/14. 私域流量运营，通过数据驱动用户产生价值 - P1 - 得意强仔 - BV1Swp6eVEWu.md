# 14. 私域流量运营，通过数据驱动用户产生价值 - P1 - 得意强仔 - BV1Swp6eVEWu

🎼企业在搭建用户体系时，一定要去做用户价值管理。如果连用户价值底层逻辑都搞不清楚，就不可能通过数据驱动，让用户产生更大价值。大家好，我是刘洋和我一起涨知识，从时间的维度出发。

我们可以把用户的价值分为过去现在和未来。基于过去产生过购买行为的用户，我们叫做老用户，而他产生的价值，对企业来说就是历史价值。这类老用户通常有两种状态，要么是活跃用户，要么是沉默用户，要想彻底激活他们。

我们要给他们一个不能拒绝的理由，要给他一个不愿意放弃的优惠政策，让他主动来消费。先激活再让他产生其他价值。通过时间维度，我们把一个用户现在即将产生购买行为的价值，称为当前价值。产生当前价值的用户可以。

🎼新用户也可以是老用户，一个新用户他往往有两种状态，要么流失掉，要么有意向购买。所以面对新用户，我们可以留下他的资料。如果当下他不购买我们的产品，我们后续可以通过资料对他进行召回。

那么如果新用户当下有意向购买我们的商品，但迟迟没有下单，大部分原因是因为价格没有达到他心理上的价格，我们可以用购物券或者实物小礼品进行提单。如果要让老用户在当下产生价值。

我们可以给他发放一些换购券或者体验券，我们把一个用户在未来产生的价值称为潜在价值。那么具有潜在价值的用户，他一定是老用户。与企业或品牌之间一定是强绑定关系的用。这类用户他可以帮我们转介绍新用户过来。

也可以再次复购我们的产品。🎼他是我们的品牌放大者，也是我们连接新用户的桥梁。那么针对企业有潜在价值的用户，我们一定要对他有相应的优惠政策，给他提供转介绍的礼品，只有把用户价值的底层逻辑理清楚。

我们才能在后续建立起用户运营模型。我今天的分享就到这里，我们下期再见。
