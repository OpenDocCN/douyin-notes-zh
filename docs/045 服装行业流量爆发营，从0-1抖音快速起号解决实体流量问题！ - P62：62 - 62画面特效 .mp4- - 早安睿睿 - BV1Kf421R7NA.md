一个是人物特效，我们先来讲讲画面特效，这里有很多app自带的特效可以选择，看一下好，我们随便点一个就能看到它预览的效果，再点一下就能调整它的一些参数，等调到你最满意的画面之后。

点确认效果就会成作为一个效果图层，放在了轨道上，点击效果图层，拖动图层的尾端，就能改变图层的长短点，效果图层我还能看到它的功能栏，它还可以直接替换成其他的素材嗯，还是很方便的。
