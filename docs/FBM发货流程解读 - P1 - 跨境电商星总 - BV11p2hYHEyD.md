# FBM发货流程解读 - P1 - 跨境电商星总 - BV11p2hYHEyD

🎼FBM发货，买好产品自己打包好，需要自己清关吗？这一步是可以给国际物流商做，还是要交给货代。做FBM的话，就是你的货从国内直接打包发到国外。正常的流程呢是这样子的啊，就是我的货啊，我去给它打包好。

然后打包好了之后呢，我要去贴上这个面单。这个面单是国际物流的面单。我在做这个面单的时候，我既要同步做好清关的工作。🎼然后面单打印出来，贴标贴好，贴好了之后交给这个货代，帮我把这个产品发出去。

这正常的流程。我们做一件代发的话呢，就是你把这个产品啊采购到货代那里，然后货代来帮你去二次打包，然后清关贴标啊，这是一个小问题，跟你的货代沟通好，就他们帮你打标的时候就要清关的。
