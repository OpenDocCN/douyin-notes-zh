最近有很多卖家莫名收到亚马逊的这封邮件，邮件内容是产品调整了收费标准。其实这个就是因为你产品的类目被更改了。有部分卖家的产品从佣金10%的类目直接被改到佣金45%的类目。

这源于亚马逊最近集中爆发的类目准确性审核。亚马逊会审核你的产品关键词嵌入等信息。如果他认为你的产品应该属于类目A而不应该属于类目B，那么它就会强制把你的产品类目给更改。类目更改之后，不但佣金会发生变化。

FB配送费可能也会发生变化。这里建议大家都赶紧查一下自己的产品收费情况。用这个亚马逊官方的FBA计算器就可以计算。只要输入一个A产品的配送成本，仓储成本佣金成本甚至利润率都可以清晰的看到。

只要在后台搜索框输入这个输入计算器，就可以使用亚马逊这个官方的计算工具了，非常方便，快速计算一下你的产品有没有被多收费吧。
