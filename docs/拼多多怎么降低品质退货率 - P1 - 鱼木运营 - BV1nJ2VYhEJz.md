# 拼多多怎么降低品质退货率 - P1 - 鱼木运营 - BV1nJ2VYhEJz

起，拼多多怎么减少品质退货率呢？把运费险关了，运费险一开羊毛挡就多了，特别是服装类穿完拍照就退货，反手再点一个品质问题退款，说衣服上有线条，你收亏不亏，那关了运费险，他没又改成仅退款了怎么办？

那你就把运费险改成7天无理由运费补贴，只有买家申请7天无理由退货才能够补贴运费。如果是品质退款就补贴不了，不仅能够有些降低品退，还能够提高你的商品利润。尤其是商品啊客单价低于30的，即使不开运费险。

也不会影响商品的转化。这么厉害，怎么设置的呀。首先还是先把运费险关了，点开消费者体验发到页面的更多停用消费者体验计划，不要选择时间，拔到最下面直接停用，然后呢，打开多多客服的消息设置。

把开场白这里啊改成这句话，这个设置之后，只要买家打开客服页面，系统呢就会自动发送这条消息给买家。如果买家想要运费险，就只能申请7天无理由了。那要是买家没看到这句话呢？

但是买家看不到就再去添加一个自定义问题，名称就写如何。😊，获取运费补贴回复改成图片，再把这张图片上传上去，这样买家点了这个问题就能自动回复这张图片了。图片展示的更加直观，也更显眼。这样设置好之后。

买家基本就能看到，还有什么不懂的，或者对拼多多运营还没有思路的，扣一，领取一份运营文档。
