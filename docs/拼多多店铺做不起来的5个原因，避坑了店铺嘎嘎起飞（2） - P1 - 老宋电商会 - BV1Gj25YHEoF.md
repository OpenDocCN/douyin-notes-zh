# 拼多多店铺做不起来的5个原因，避坑了店铺嘎嘎起飞（2） - P1 - 老宋电商会 - BV1Gj25YHEoF

对拼多多卖家来说，最怕的就是用了各种办法，店铺还是做不起来。一般来说，拼多多店铺做不起来的原因有5个。今天继续昨天的分享说下后面两个原因。第四个原因是活动难。随着商家数量大量增长，市场竞争变得更加激烈。

加上消费者对于产品的要求也越来越高。拼多多活动小二在选品时面临的选择就更多了。他们自然是选择品质更好，价格更合适的商品来参与活动。加上平台上还有大量KA商家，也就是平台重点客户会直接跟平台签订协议。

这些商家的广告投入往往很大。自然可以获得平台活动资源位以及更多自然流量扶持。与此同时，活动报名对商家实力，店铺和商品的要求也越来越高，并对店铺和商品的日销售额数据有要求。

从而导致大部分中小卖家报不上活动，其实拼多多上面的活动有很多，适合新品的活动也不少，只是不同活动有不同的报名要求。第五个原因是效果差。拼多多活动的考核机制越来越成熟，精准和严格。

现在很多活动都是按照时间段的产值来分配铺光量的，比如秒杀都是按照每分钟产值来分配铺光量，还有很多活动按照每小时的产值来分配铺光量。同时，拼多多推出千人千面的流量逻辑，加重推荐式流量以及流量碎片化的状态。

这些都让活动的效果越来越差，更重要的是活动效果好不好。主要考核的标准是UV价值计算公式，UV价值等于总访客比总营业额。
