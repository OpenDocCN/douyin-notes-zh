# 商品前期做好增量计划，才更容易爆单 - P1 - 夜老板电商 - BV1m6sUeTEau

如果你想要你的链接能够爆单，一定要在前期就学会自己去制定计划，一定要让你链接的销售额每天都在增长。销售额逐步增长，这个指标是很重要的一个权重指标。如果前期没有完成，这个链接它的上限就会非常低。

就很难真正在后期去爆单。举个最简单的例子，你们在用直通车去起链接的时候，基本上都会发现稍微好点的链接，对吧？开车它每天的单量其实是逐步在上升的，成交金额也是比如第一天卖了200块钱，第二天卖了500。

第三天卖了700，它是不是会有这样一个上升的过程。但是呢很多很多链接，它都会达到一个瓶颈。比如说你做了一个星期两个星期，它每天固定的金额它上不去了，它就只能在1000左右徘徊。今天900，明天1100。

后天1000对吧？这个是链接本身它都逐步增长达到一个上限了。这个时候如果你不去管他，就让他这样去跑，那你这个链接基本上啊它的门槛也就到这儿了。后面也很难再上去了。甚至后面你再去调投产，一调投产。

这个链接就掉量，或者直接断流死掉。所以说一定要在前期起链接的时候去给自己制定计划。比如你今天卖了1000，明天就要达到1500，后天就要达到2000。如果达不到，就要人为的去干预，去想办法。

在前期一定要一让链接的成交金额去增量。你可以补单，可以去设优惠券，可以去固定优惠券的数量，张数去冲量，可以用限量购设置固定的库存，对吧？价格低一点去冲量去拉成交金额，可以去报限量的活动。

包括呢走多多进宝也是可以的。反正用尽各种手段，一定要制定一个交易额的成长计划，就像我刚才跟你讲的，我今天1500，明天就一定要2000，后天就一定要2500，以此类推。

最起码在前半个月你要保持这样的一个增长。你的链接好的话，那他报的时候就一定自己会报。如果说半个月你这样做完，它都报不起来，那这个链接它本身可能就不是一个特别优秀的链接，能做到什么极限。

就做到什么极限就可以了。所以前期的时候一定要这样去给自己制定一个计划。那么你在有好的链接的时候，才能够真正的让它起报。
