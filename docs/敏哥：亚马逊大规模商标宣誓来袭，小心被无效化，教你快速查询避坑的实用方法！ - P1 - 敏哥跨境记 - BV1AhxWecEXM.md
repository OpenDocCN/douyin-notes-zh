# 敏哥：亚马逊大规模商标宣誓来袭，小心被无效化，教你快速查询避坑的实用方法！ - P1 - 敏哥跨境记 - BV1AhxWecEXM

🎼大家要注意啊，兄弟们，商标不是你注册完了就结束了，每5年左右是要做一次宣誓的。如果你不做宣誓，商标就会被无效，搞不好还会被人抢住，反过来投诉你侵权。所以要核实好自己名下所有商标的宣誓截止时间。

最好复制记录在备忘录中，设置好提醒，一旦到了可以宣誓的时间，立马安排去做宣誓。具体怎么查截止时间呢？打开商标局官网，找到treread max，点击下方第一个按钮，接着点击商标搜索系统。

输入商标名称或者注册号，点击记录后找到menttenance按钮，点击后就可以看到三个时间。第一个时间是可以提交宣誓的时间。第二个时间是最晚免费提交宣誓的时间。最后一个时间是付费提交宣誓的截止时间。😡。

🎼这三个时间点就是你商标要宣誓的三个重要的时间点，尽量在第一个时间跟第二个时间点之间去完成商标宣示的操作，以免带来一些潜在风险。🎼做药训就是这样，杂式和细节很多，无论到什么时候，风控就应该放在第一位。
