# 2. 搭建用户运营体系，用RFM模型做用户精细化运营 - P1 - 得意强仔 - BV1uwp6e5ELZ

🎼有很多企业在搭建运营体系时不知道该使用什么样的模型。那么我推荐你使用RFM模型，它可以相对精确的判断一个用户的长期价值甚至是终身价值。大家好，我是柳洋，和我一起涨知识。

RFM模型是衡量客户价值和客户创造利益能力的重要工具和手段。主要通过用户行为来区分用户价值，也是使用最广泛的一个模型，它是以三项指标来描述客户的价值状况。

主要以一个用户最近一次的消费行为、消费频率、消费金额来进行判断。我们来讲第一个指标，R最近一次消费。用户消费的过程是持续变动的。那么我们这个数据在顾客上一次消费满一个月后。

有可能在下一个月它就成为两个月以上的客户了。如果用户最近一次消费的行为数据在不断的增加。🎼那么企业它是成稳步成长的一个状态。如果这个数据越来越少，说明企业有走下坡路的征兆，所以我们在建立模型时。

一定要做好R值的数据宗值。第二个指标F消费频率是顾客在限定期内所购买的次数。那么用户消费频率越高，也就说明对我们的满意度越高。我们增加用户购买次数就意味着从竞争对手中抢占市场占有率。

我们通过数据分析可以把消费一次的用户变成消费两次的用户。长此以往，我们就可以把一个新用户变成我们的忠实用户。在企业里80%的收入是来自20%的重点用户。理论上，F值和M值，他们是都具有一定的时间范围的。

像通常是一年，所以我们要找到在一年内消费金额比较高的用户来进行重点。🎼维护，这也是RFM模型当中的第三个指标M消费金额。给大家举个例子，如果你现在搞活动，给1万个人发活动信息。

但是最终到店的转化只有1%。如果你通过RFM模型来分析，找到R值最高的用户，也就是说，最近一次到店来购买的用户，你给他们发信息，最终到店转化率可能从1%直接升到10%。

我们给1000个人发信息和给1万个人发信息的宣传费用是不同的。不仅节省了宣传费用，而且转化率也大大的提高了，这就是通过RFM模型分析用户行为找到关键用户来进行精准的推送。
