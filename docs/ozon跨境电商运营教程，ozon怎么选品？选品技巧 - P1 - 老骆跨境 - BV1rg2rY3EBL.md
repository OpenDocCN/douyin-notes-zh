# ozon跨境电商运营教程，ozon怎么选品？选品技巧 - P1 - 老骆跨境 - BV1rg2rY3EBL

🎼欧众快速出单的选品技巧，我们都知道，跨境电商就是七分选品，三份运营，一旦选品出了问题，那花再多的时间在运营上其实也都是无用功。来点个赞。接下来我来详细讲讲具体该怎么做。

首先我们可以先在欧众上选择一个类目，最好是那种二三级的小类目。因为这种小类目一定存在着大量的南海市场，然后再通过这个工具把里面的商品销售数据全部扒下来。最后我们就会得到这样一个表格。

然后我们再把表格中销量和排名靠前的留下来，最好是选择那种连评论都没有。但是销量还不错的品，然后把这些品全部上架到自己的欧众店铺去。通过这种方式，我们就能挖掘到大量的细分类目的南海产品怎么样？你学会了吗？
