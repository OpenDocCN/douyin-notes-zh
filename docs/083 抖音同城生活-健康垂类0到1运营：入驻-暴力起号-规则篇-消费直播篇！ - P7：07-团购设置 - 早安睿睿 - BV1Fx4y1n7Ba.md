# 083 抖音同城生活-健康垂类0到1运营：入驻-暴力起号-规则篇-消费直播篇！ - P7：07-团购设置 - 早安睿睿 - BV1Fx4y1n7Ba

![](img/1c74667ab17e6083b911735a2cb863f4_0.png)

本节课程我们将为大家演示如何设置团购。

![](img/1c74667ab17e6083b911735a2cb863f4_2.png)

打开抖音来客app，进入抖音来客首页。

![](img/1c74667ab17e6083b911735a2cb863f4_4.png)

第一部分，创建团购商品，第一步登录账号。

![](img/1c74667ab17e6083b911735a2cb863f4_6.png)

会看到已经完成入驻流程的页面，点击商品管理，点击创建商品，即可创建新的团购商品，点击创建商品，选择商品的品类，商品品类可以选的范围，是您门店经营品类的集合，选择商品类型。



![](img/1c74667ab17e6083b911735a2cb863f4_8.png)

团购指的是，例如商家上架一个200块钱的双人套餐，里面包含了很多菜品，这个属于团购的套餐，代金券指的是例如80元抵100元代金券。



![](img/1c74667ab17e6083b911735a2cb863f4_10.png)

第二步点击下一步会出现具体配置页面，收款方式有两种。

![](img/1c74667ab17e6083b911735a2cb863f4_12.png)

一个是统一收款，一个是分门店收款，统一收款指的是商品上架之后，所有核销的款项都会统一打入公司收款账户，分门店收款指的是，如果已经有门店绑定了收款账户，那么这个门店核销的款项，就会进到分门店收款账户中。

没有绑定收款账户的门店，核销款项会自动进入公司收款账户中。

![](img/1c74667ab17e6083b911735a2cb863f4_14.png)

点击试用门店，显示已经认领或者授权的门店，直接进行点选，返回刚刚的页面，在商品详情下方输入自定义套餐名称，选择就餐人数。



![](img/1c74667ab17e6083b911735a2cb863f4_16.png)

进入商品搭配环节进行套餐，商品填写这边的价格指的是单品的原价。

![](img/1c74667ab17e6083b911735a2cb863f4_18.png)

选择可选范围设置。

![](img/1c74667ab17e6083b911735a2cb863f4_20.png)

即选几，用户下单到店去消费时，足品里边如果是二选一，说明这两个商品当中用户只能选择一个。

![](img/1c74667ab17e6083b911735a2cb863f4_22.png)

然后点击保存，顾客实际需支付，指的是顾客实际需要支付的价格，在商品照片栏完成商品照片的上传。

![](img/1c74667ab17e6083b911735a2cb863f4_24.png)

选择投放渠道，一个是不限制商品上架之后可以全渠道售卖，另一个是进直播间售卖。

![](img/1c74667ab17e6083b911735a2cb863f4_26.png)

只有在直播间挂了这个商品，并且在开播的时候，用户才可以购买。

![](img/1c74667ab17e6083b911735a2cb863f4_28.png)

其他的情况下都不可以售卖，选择售卖日期，设置顾客消费时间限制，第三步点击提交审核。

![](img/1c74667ab17e6083b911735a2cb863f4_30.png)

审核规则。

![](img/1c74667ab17e6083b911735a2cb863f4_32.png)

第二部分管理团购商品，温馨提示，多门店经营场景，目前仅品牌户账号有操作商品权限。

![](img/1c74667ab17e6083b911735a2cb863f4_34.png)

门店账号仅可查看使用于此门店的商品情况，无操作权限，若需操作，请切换至品牌户账号登录。

![](img/1c74667ab17e6083b911735a2cb863f4_36.png)

第一步进入app首页，点击上方商品管理，进入团购活动列表页，可对已有团购活动进行操作，第二步一对已上架进行中的团购进行编辑，更多编辑，编辑完成之后需要提交审核。



![](img/1c74667ab17e6083b911735a2cb863f4_38.png)

二对未通过的团购商品，需要重新编辑并提交审核，三已下架商品需要重新编辑上架，四，草稿箱中的团购商品需要重新编辑并提交审核。



![](img/1c74667ab17e6083b911735a2cb863f4_40.png)

特别注意已上架团购商品设置延长售卖日期。

![](img/1c74667ab17e6083b911735a2cb863f4_42.png)

团购延期已上架，团购商品设置下架结束售卖，审核团购商品若签署代运营协议，则增加审核，团购商品页面，由抖音的业务人员帮忙发布的团购商品，在正式发布前需要商家帮忙审核，商家可以查看并进行审核。



![](img/1c74667ab17e6083b911735a2cb863f4_44.png)

第三部分推广团购商品。

![](img/1c74667ab17e6083b911735a2cb863f4_46.png)

一如何通过短视频推广团购商品，商家绑定抖音号，且发布了团购商品后，通过与其绑定的抖音号在抖音上发布短视频，发布短视频时可添加门店或团购商品，仅可以添加该商家认领的门店。



![](img/1c74667ab17e6083b911735a2cb863f4_48.png)

及发布的团购商品，视频发布后，用户在该视频页面左下角。

![](img/1c74667ab17e6083b911735a2cb863f4_50.png)

可以看到关联的门店或团购商品信息。

![](img/1c74667ab17e6083b911735a2cb863f4_52.png)

二如何通过直播推广团购商品，一打开抖音app。

![](img/1c74667ab17e6083b911735a2cb863f4_54.png)

点击最下方加号，二选择开直播，三，点击团购组件可查看或编辑已组或完成的商品，四点击开启视频直播即可开启直播。



![](img/1c74667ab17e6083b911735a2cb863f4_56.png)
