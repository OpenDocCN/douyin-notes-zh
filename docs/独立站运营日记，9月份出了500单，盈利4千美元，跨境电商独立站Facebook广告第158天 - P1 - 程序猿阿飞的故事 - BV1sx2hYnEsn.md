# 独立站运营日记，9月份出了500单，盈利4千美元，跨境电商独立站Facebook广告第158天 - P1 - 程序猿阿飞的故事 - BV1sx2hYnEsn

🎼大家好，我是阿飞。今天是我做独立站的第158天。最近整理了9月份独立站的数据分享给大家看看。9月份两个shoppas独立账一共出了500单，累计销售额19591。27美元，目标一天30单。

这个数据只晚谈了目标的55%。😊，🎼整个这个月波动比较大，最好的一天34单，最差的一天两单，平均一天16单左右左右，目前签收金额9795美元。🎼飞bo广告花费2330美元。

物流成本和采购成本是3000美元左右，利润是4475美元左右，是今年最好的一个月，是丰收的一个月。但是后续心里完全还是没有底。🎼9月份测出了6个左右的小爆品，测评成功率真的是玄学。

还没有摸索出很稳定的经验。

![](img/06d1c5832653b65027d0d48398682b66_1.png)

🎼沉默成本也挺高的，10月份开始是跨境电商旺季。🎼自己资金有限，抗风险能力比其他电商团队差很多，偶尔还是有很大焦虑。9月份一周跑三次5公里和游两次泳来减压，目前运动才能让自己真正快乐。
