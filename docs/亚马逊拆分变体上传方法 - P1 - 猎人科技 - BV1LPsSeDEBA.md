# 亚马逊拆分变体上传方法 - P1 - 猎人科技 - BV1LPsSeDEBA

🎼亚马逊的定制计划，保姆式教学来了，大家快看商品规定部分区域是由买家定制，建议提前设好模板。首先打开指纹官网，在右边搜索框这边搜索你想要的商品，然后点击你要定制的商品，进入产品页面后。

点击底下蓝色方框的定制设计，在设计页面里进行设计，设计好后，点击右上角的保存，然后点击下的效果图，然后打开亚马逊管理库存页面，在后面编辑这边，打开下拉框，选择添加编辑自定义信息。

进入页面后添加刚刚设计的表面，等页面浮现出来，添加产品的标签说明，上传产品的预览图，接下来点击添加定制，选择颜色或者点击底下的选项下拉框菜单，修改一下定制项的名称，视先移到底下表格这边选项名称。

这边是买家看到的名称，可以输入产品颜色的名称。如果你有很多颜色选项的话，点击右下角这边的添加多个选项就好了。查下这边可填可不填设置好后，我们看一下预览图，效果大概是这样的。😊，🎼差不多就好了。
