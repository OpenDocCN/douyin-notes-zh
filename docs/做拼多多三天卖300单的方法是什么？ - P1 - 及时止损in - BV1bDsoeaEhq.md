# 做拼多多三天卖300单的方法是什么？ - P1 - 及时止损in - BV1bDsoeaEhq

🎼拼多多三天卖百单的方法是什么？第一天做好百里图长途和场景图，第二天用数据资产管理截取竞品流量。第三天原价报名一个秒杀活动，3天破百单，轻轻松松。具体操作是什么呢？

第一天的百里图可以去商品素材里面制作长图呢可以直接用别人的详情页的图。场景图就是你的副能播图跟主图只要不一样就行。第二天的操作啊，只要去后台的推广工具里面，打开数据资产管理，输入产品的关键词。

添加竞品数据，最后复制竞品的人群和流量数据，在管理渠道投放3到5天就能够轻松产生它的流量。第三天，原价报名秒杀可以看我9月20号的原价上秒杀的视频。原理和操作思路啊都讲的很清楚。
