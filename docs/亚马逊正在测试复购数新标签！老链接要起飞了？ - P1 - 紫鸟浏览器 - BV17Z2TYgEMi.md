# 亚马逊正在测试复购数新标签！老链接要起飞了？ - P1 - 紫鸟浏览器 - BV17Z2TYgEMi

🎼亚马逊移动端正在进行灰度测试，老链接要起飞了。现在在部分买家的移动端搜索结果页面中，有链接的销量标签不再是显示上月购买了多少数量，而是显示产生了多少次再次购买。

例如这个bests seller链接在4。6星级评分下边就显示有100万件不购。不过目前并非所有用户的搜索结果页面都会显示复购数量，同一个搜索词，同一个搜索结果页，有的用户还是显示上月购买数量。

但不难看出，亚马逊目前似乎有一点复购率高的产品更多的展示。要知道，消费者如果看到这个标签，相当于看到其他消费者对这个产品的认可，因为只有认可才会复购。因此，购买转化率可能也会大幅提高。

对于有高复购的老链接，如果显示这个标签分分钟就是现金奶牛维持难度也是事半功倍，但对于毫无销量的新链接来说，可能是一次不小的冲击，卖家们要做好准备。😊。
