# 拼多多，一周只卖一单正常吗？ - P1 - 凯文电商 - BV13QxyeTExC

做多多一周只卖出去一栏正常嘛？肯定是你只上架了个链接，其他的什么都没有去做链接没有权重，一单不卖，这都是正常的。如果你想查看权重，就打开店铺的流量数据，点击搜索流量。

在这儿你能够看到整体商品搜索指数的一个曲线图。如果曲线低于同行平均水平或者直接躺平的一个状态，就是权重太低了。像你赶紧去补指数拉高权重。如果说你不知道怎么补权重的，就打开你的流量看板。

把你店铺访客数和UV价值的同乘均值给它记录下来，把这两个数值相乘，就是我们需要做的坑产。然后呢，用这个数除以你的客单价，就是你需要做的单量了。去做一周指数呢会明显的有提升。如果你还有什么不懂的。
