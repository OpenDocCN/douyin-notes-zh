# 敏哥：跨境电商老板最常见的管理误区，绩效考核的目的很多人都搞反了！ - P1 - 敏哥跨境记 - BV1EE2RYGEuF

🎼其实我很少会主动来管理，但总是感觉啊大家对管理存在很多误区和偏见，要么就是压根认为管理不重要，不需要管理，要么就是走了另一个极端，到处学各种管理方法和考核制度。学会来应用后，反而把团队搞得一团糟。😡。

🎼在我看来，10人以下的跨境电商团队管理能起到的效果并不大，或者说小团队的管理跟大团队是有本质区别的。😡，🎼10人以下小团队的管理重心更应该要放在做事上，管理人是工作的流程和细节。

一遍遍的优化细节调整方法，直到最后大家都能拿到结果，拿到更多的钱。😡，🎼而规模比较大的团队的管理相对更重要一些。在做到上述要求的同时，要辅助一部分的绩效考核和制度，不然很容易就乱套。😡。

🎼但是很多管理者或者老板一提到绩效考核，想到的就是提各种各样的要求，达不成，不管三加二一就扣钱。😡，🎼其实我们设计任何的考核指标之前，都需要仔细思考一个问题。这个绩效搞完之后。

员工能拿到手的钱是变多了还是变少了？😡，🎼如果你作为老板给员工定了各种各样的指标，提了各种各样的要求。但是最后员工的工资合同完之后却减少了。你觉得你的要求能执行落地吗？很难。😡。

🎼绩效考核的目的不应该是为了扣工资，而应该是涨工资。我们设定绩效的目的一定是要起到激励的效果的，一定是要能让员工通过更细节的工作，更强的执行力，达到绩效目标，创造更多利润量的同时，自己也能拿到更多的钱。

这样正向的激励效果才能出来，激励效果一旦出来记入证据完之后，很多小细节就不需要你操心了。员工自己甚至能考虑到你原先没有注意到的细节点。😡，🎼考核永远不是目的，拿到结果才是你觉得呢？😡。
