# 天猫禁止引流私域，商家该如何应对？ - P1 - 数字营销攻略 - BV14S411N7pG

近日，天猫对诱导第三方规则进行了调整，明确禁止了商家包裹卡引流到其他平台的行为。虽然没有明说是哪些平台，大家也知道指的是威胁新规的实施意味着，从7月1日开始。

商家以往通过包裹卡进行的私域引流行为将不再被允许。不同程度违规的商家也会受到严厉的处罚措施。对商家的影响。早期，入驻电商平台的商家少，竞争不激烈，流量成本低。可是当越来越多的商家涌入，流量趋于稳定。

电商获客成本逐步攀升，红利逐渐消失。如何获得更低成本流量，成为各大商家亟待解决的问题。破局方法之一，就是搭建私域流量，商家在电商平台花钱投广告，吸引用户下单，然后通过包裹卡引流到私域。

进行后续提升复购的操作，这已经成为了一种非常成熟的玩法，对提升ROI降低营销费用有着关键的作用，对很多电商从业者来说会是一个打击，意味着他们将失去一个公域引流用户的渠道，也将失去一条与用户建立联系。

提供服务，提升复购的道路，尤其是对那些在私域引流上重度依赖包裹卡的商家会是一个致命打击，他们或会因为新规，失去私域用户新增量。商家如何应对？对商家来说，只依赖单一引流手段风险极大。

要多去探索更多元、更合规的营销方式，寻找新的增长点和出路，才是未来。对于有实力的企业，不仅有技术，而且还懂推广和运营的话，可以考虑自建APP或独立站，彻底摆脱对平台的依赖。

把整个生意牢牢的掌握在自己手中，布局新的流量平台，比如视频号小店，目前，腾讯正在大力发展视频号，被看作是整个腾讯的希望，视频号小店也处于发展的初级阶段，还是有很大潜力和红利机会的。

而且视频号小店的包裹卡，还是可以引流的，有一些可替代的引流方案，可供参考，比如可以考虑在产品外包装、快递外包装、社保卡、说明书等放置二维码，再比如引导客户在品牌天猫店铺入会，扩充品牌公寓会员池。

然后再将会员资产引流到私域。😊。
